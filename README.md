# Goodreads Books Data Analysis & Interactive Dashboard (Tableau)
## Project Overview
This project explores a Goodreads dataset using data visualization techniques in Tableau to uncover patterns in books, authors, publishers, and reader engagement.
The goal was to transform raw book data into an interactive dashboard that answers key questions about publishing trends, popular authors, and reader preferences.
## Objectives
	- Analyze book distribution across authors, publishers, and years
	- Identify top-performing books based on ratings
	- Explore language distribution in published books
	- Build an interactive dashboard for dynamic filtering and insights
## Dataset
	- Source: Kaggle
	- Key fields used:
	  - Title
	  - Author(s)
	  - Publisher
	  - Publication Date
	  - Language Code
## Dashboard Features
1. Authors by Total Books (Bar chart)
  - Stephen King and P. G. Wodehouse lead with 40 books each
Insight: Highly prolific authors dominate dataset representation
2. Publishers by Total Books (Bar chart)
	- Vintage: 318 books
	- Penguin Books: 261 books
Insight: A few major publishers account for a large share of book production
3. Titles by Ratings Count (bar chart)
	  - Twilight leads with 4,597,666 ratings
Insight: Popularity is heavily skewed toward blockbuster titles
4. Books by Publication Year (Area Chart)
	- Peak year: 2006 with ~1,700 books
Insight: Indicates a surge in publishing activity or dataset concentration
5. Language Distribution (Pie Chart)
	  - English (eng) dominates with 8,908 books
Insight: Dataset is heavily English-centric, which may influence analysis bias
6. Interactive Text Panels
Dynamic fields update based on selected title:
	- Author(s)
	- Publication Date
	- Language
	- Publisher
	- Text Reviews Count
	- Average Rating
	- Ratings Count
Feature Highlight:
Users can filter by title and instantly see all related metadata change across the dashboard.
## Key Insights
	- Popular books receive disproportionately high engagement
	- English is the primary language across most entries
	- Publishing activity peaked around mid-2000s
